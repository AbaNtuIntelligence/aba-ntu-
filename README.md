# AbaNtu Intelligence

## Know Thyself. Master Thyself. Align Thyself. Build Thyself.

A personal development platform integrating **Psychology**, **Astrology**, and **Etymology** for holistic self-mastery.

## Features

- 🧠 **Self-Mastery Assessment** - 50-question assessment across 10 dimensions
- ✨ **Astrological Integration** - Personalized insights based on birth date
- 📖 **Etymology Journal** - Track language patterns and reality construction
- 👤 **Personalized Dashboard** - Tailored content based on user data
- 🔐 **Supabase Auth** - Secure authentication and profile management

## Tech Stack

- Next.js 16 (Turbopack)
- TypeScript
- Tailwind CSS
- Supabase (Auth + Database)
- Zodiac Astrology Logic
